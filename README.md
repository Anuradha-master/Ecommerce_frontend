# 🛒 E_commerce_Frontend

A responsive and user-friendly E-commerce frontend website developed using **HTML, CSS, and JavaScript**.  
This project provides essential e-commerce functionalities such as product listings, cart management, login pages, admin pages, and localStorage-based data handling.

---

# 🚀 Features

- 🛍️ Product Listing Page
- 🔍 Product Details View
- 🛒 Add to Cart Functionality
- ❤️ Wishlist Support
- 👤 User Login & Signup
- 🔐 Admin Login
- 📦 Orders Section
- 💾 localStorage Integration
- 📱 Responsive User Interface

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage
